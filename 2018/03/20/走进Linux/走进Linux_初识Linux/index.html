走进Linux_初识Linux

发表于 2018-03-20 | 分类于 走进Linux | 本文总阅读量 次

引：了解Linux，学习Linux，深入Linux。

Linux是什么

Linux就是一套操作系统，它的内核是Linux Torvalds在1991年开发出来的。

Linux的内核版本

看下面的例子：

  1. 主、次版本为奇数：开发中版本，如2.5.xx，这种版本仅内核开发工程师使用。
  2. 主、次版本为偶数：稳定版本，如2.6.xx ，家庭、企业使用。

我们可以通过下面的命令查看自己的linux的版本号：

uname -r

Linux发行版

Linux其实就是Linux Kernel（内核）＋内核工具。他是GNU GPL授权模式，任何人都可以获取源代码并进行修改。因此Linux的全名是：GNU/Linux。
Linux ditribution是由各个商业公司开发，本质上是LinuxKernel＋内核工具＋软件，让普通用户都能使用的操作系统。 基本上Linux distributions并无太大区别，可以根据自己的喜好选择。他们的内核都是从www.kernel.org上下载的。

Linux的特色

  1. 自由与开放的使用与学习环境
  2. 配置需求低廉
  3. 内核功能强大而稳定
  4. 独立作业

Linux优点

  1. 稳定的系统
  2. 免费或少许费用
  3. 安全性、漏洞的快速修补
  4. 多任务、多用户
  5. 用户与用户组的规划
  6. 相对比较不耗资源的系统
  7. 整合度佳且多样的图形用户界面

Linux缺点

  1. 没有特定的支持产商
  2. 游戏的支持度不足
  3. 专业软件的支持度不足

Linux当前的角色

企业环境的应用

  1. 网络服务器（当今最热门的运用）
  2. 关键任务的应用（金融数据库、大型企业）
  3. 学术机构的高性能运算任务

个人环境的使用

  1. 桌面计算机(在Linux系统上装一个X Window System软件之后，就能有桌面了)
  2. 手持设备（PAD、手机）
  3. 嵌入式系统

Linux的学习路线

  1. 学习计算机概论及硬件相关知识
  2. 从Linux的安装开始学起
  3. 学会使用Linux的基本技能
  4. 学会vi编辑器
  5. shell与shell脚本的学习
  6. 一定要会软件管理员
  7. 了解网络基础
  8. 使用Linux架设网站

总结

在使用Linux之前请学会Linux！
